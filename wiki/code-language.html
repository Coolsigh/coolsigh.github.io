中文使用说明

利用真·符号语言(以下简称"符语")表示中文时, 须按照标准汉语拼音规则来进行拼写. 因此, 若要使用符语来表示中文, 你必须完全精通汉语拼音规则.

音调

与标准汉语拼音不同的是, 符语的中文音调不标在韵母上,而是标在音节之后.

以下是一个范例:


你好啊! → nǐ hǎo a ! → niv haov a= ! → Λ⊥v├Π▷vΠ=!


注: 该范例最后的"啊"为轻声, 用符语音调表示时用"=".

连写

在利用符语表示中文时, 经符语表示后的文字可单个音节分开写, 也可以连在一起写.

以下是一个范例:

你好啊!

·Λ⊥v ├Π▷v Π=!
·Λ⊥v├Λ▷vΠ=!

如果连在一起写, 则需以音调位置来分出单个音节. 以下是一个范例:


Λ⊥v├Π▷vΠ=!

Λ⊥v ├Λ▷v Π=!

即使分写连写都可以, 但对于使用符语较为熟练的人(3级及以上)而言, 我们提倡且应该使用连写; 对于初学者或不太熟练的人(2级及以下)而言, 我们则提倡分写, 但鼓励尝试连写. 在运用达到较熟练的程度时, 则应改为连写.


阿拉伯数字和英文字母

由于符语中存在部分字符与阿拉伯数字和英文字母易混或有所冲突, 特制定以下规则.

数字

在符语中使用数字时, 必须在其前后空出足够的距离, 以便于分辨. 数字无需变形, 直接代入即可.

以下是一个范例:

ʃ＜/▽＜v╛⊥\ 5 日ΠΛv.

若数字中含有符号(如小数点、百分号、分数线等), 无需变形, 直接代入即可.

字母

在符语表示中文时若带有字母, 则必须在其前后空出足够的距离, 以便于分辨.

字母可以不作变形直接代入, 也可以转化为符语代入. 但如果转化为符语代入, 则其前后不仅要空出足够的距离, 还要在其前后加上一对大括号{}.

以下是一个范例:

Δ＜ΠΛv D ྈ├ΞΛ◁\>＜Ξ\. **或者** Δ＜ΠΛv {╛} ྈ├ΞΛ◁\>＜Ξ\.

有关于 下划线 的使用请参照 英文使用说明

单个字/词、生僻字/词、书面语

利用符语表示中文时, 会存在多个汉字共用一种拼写方法的情况. 此时需要根据上下文语境来判断该音节所代表的汉字. 因此, 我们不建议利用符语来表示单个字/词, 且很不建议利用符语表示生僻字/词及书面语, 以免产生混淆和歧义. 应简明易懂.

但如果实在无法避免以上情况, 也没有更好的方法; 目前我们正努力解决该问题.

语音变调

在汉语中, 常有语音变调的情形. 因此在使用符语表示此种汉字时, 应按照变调后的语调拼写.

以下是一些范例:

不(bú)要 - 日＜/▽Π▷\

不(bù)能 - 日＜\ΛΞΛ◁/

一(yí)个 - ▽⊥/◁Ξ\

一(yì)条 - ▽⊥\+⊥Π▷/

十一(yī) - ~├⊥/▽⊥-

英文使用说明

利用符语表示英文相对于表示中文而言更加简单且更加便捷. 由此可知符语更适用于英文.
以下是关于符语表示英文时的使用说明.

利用符语表示英文时, 有关于语法上的规则无需做出更改.

大写字母

在英语中, 字母常有大写的情况. 为此, 我们制定了一套适用于符语的字母大写规则, 以符合英语语法规定.

在符语中的大写字母, 是在其已转化为符语的前提下在其下加上下划线. 如

YES! → ▽Ξ~!
Hello, Tom! → ├ΞΓΓ▷, +▷ɜ!

数字和字母

英文中数字和字母的表示规则相对于中文而言会比较简单.

数字

规则同中文.

字母

在英语中, 也常遇见用单个字母来表示序数义的情形. 此时仅需将其转化为符语即可, 不需添加任何字符.

严禁连写

用符语表示中文可以连写, 但若用英语, 则绝对禁止连写, 否则轻则造成歧义, 影响理解, 重则可能造成不良影响!

外来字母

在英文中, 有时会用外来字母表示专有名词. 此时无需对外来字母进行变形, 直接将其代入即可.

注意事项

对于该语言的初学者而言, 有以下内容需注意.

适用场景

符语仅适用于纸面体现.

不适用于正式场合.

可作为密语使用.

学习提示

学习时长

1天到1星期.
请不要心急, 任何事物都是有过程的.

易混警示

符语中有很多符号都互相易混, 望区分仔细!

后缀说明

后缀说明相当于前面提到所有使用方法中共通用的说明. 以下是一些后缀说明:

标点符号

在本手册中提到过的所有使用说明都通用以下内容:

标点符号直接代入即可, 无需进行变形. 但请尽量不要使用与符语符号易混或发生冲突的符号.

发音

在本手册中提到过的所有使用说明都通用以下内容:

符语没有发音. 创立者原本想创立一个符语发音体系. 因精力不足, 且此体系欲实现显然不现实, 因而未创立发音体系.

关于符语

符语是创立者在闲暇之余创立的, 同时也创立了一个较为完整的体系. 但符语从严格意义角度来说并不能算作一种真正的语言, 仅是将字母换了一个形状而已.
创立者创立符语的目的只是为了写字方便和作为密语使用.
